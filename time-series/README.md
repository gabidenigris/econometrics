# Time Series

Lecture notes and reproducible R code for an applied time series econometrics course, taught by Prof. Silvio Costa. The material moves from the statistical foundations through stationarity, ARIMA modeling, volatility, and multivariate dynamics.

Each lecture comes in two files:

- **Lecture (PDF):** the full notes with theory, formulas, and plots. Opens and renders directly here on GitHub.
- **Code (.Rmd):** the R Markdown source with all the code. Download it to reproduce every result.

Examples pull live data from Brazilian public APIs (Banco Central via `rbcb`, IPEADATA via `ipeadatar`), so most scripts run end to end without local data.
